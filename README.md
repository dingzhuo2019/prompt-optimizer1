# 🚀 Prompt 优化器

一个简单实用的网页小工具，帮助产品经理和运营人员优化 AI Prompt，让 AI 更精准地理解需求。

## 📖 功能特点

- **任务类型选择**：支持通用任务、文案写作、数据分析、代码开发、设计方案、营销策划、行业研究等
- **语气风格定制**：专业严谨、友好亲切、简洁明了、详细全面、创意活泼
- **输出格式指定**：纯文本、Markdown、表格、列表、JSON
- **一键复制**：优化结果可快速复制到剪贴板
- **使用小贴士**：内置 Prompt 编写最佳实践建议

- 产品经理
- 运营人员
- 市场策划
- 内容创作者
- 任何需要使用 AI 工具提升工作效率的人

## 🚀 快速部署

### 方式一：GitHub Pages（推荐）

1. 将此项目上传到 GitHub 仓库
2. 进入仓库 Settings → Pages
3. 选择分支（main/master）和根目录
4. 等待几分钟后，访问 `https://你的用户名.github.io/仓库名/prompt-optimizer.html`

### 方式二：Vercel / Netlify

1. 将代码上传到 GitHub
2. 在 Vercel 或 Netlify 导入仓库
3. 自动部署完成

### 方式三：本地使用

直接双击打开 `prompt-optimizer.html` 文件即可在浏览器中使用

### 方式四：任意静态托管

将 `prompt-optimizer.html` 上传到任何支持静态文件的服务器或 CDN

## 💡 使用示例

**原始 Prompt：**
```
帮我写一份产品文档
```

**优化后：**
```
【角色设定】
你是一位在文案写作领域的资深专家，拥有丰富的实战经验和专业知识。

【任务背景】
帮我写一份产品文档

【具体要求】
1. 请以专业严谨的语气风格进行回答
2. 输出格式要求：Markdown 格式
3. 确保内容准确、逻辑清晰、结构完整
4. 如有必要，请提供具体的案例或数据支持
5. 避免使用过于专业的术语，确保内容易于理解

【输出规范】
- 开头简要说明核心观点
- 主体部分分点阐述，层次分明
- 结尾总结要点或提供行动建议
- 如需引用数据或案例，请注明来源或说明假设条件

请根据以上要求，为我提供专业、实用的解答。
```
